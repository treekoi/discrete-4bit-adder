# CD4000 4-Bit Binary Adder PCB

> **🚧 Status: Work in Progress**
>
> This repository contains the design files for a custom 4-bit binary adder built using discrete CD4000 series CMOS logic ICs.
> The hardware has been physically assembled and powers on via a 9V battery, but the PCB routing is currently undergoing final error-checking and optimization.
> Full schematic, PCB layout, and BOM files will be released shortly.

## Project Overview

Designed as part of the **Digital Logic Workshop at the University of Alabama at Birmingham (UAB)**, this project demonstrates the fundamental principles of binary addition using discrete logic gates rather than microcontrollers or FPGAs.

**Key Features:**

- **Logic Family:** CD4000 Series CMOS (e.g., CD4008, CD4070, etc.)
- **Input:** 8x DIP Switches (4 bits per operand + Carry In)
- **Output:** 5x LEDs (4-bit Sum + Carry Out)
- **Power:** Onboard regulation for 9V Battery input
- **Control:** Rocker switch for power management
